# 夜莺不语 网页规划

## 需求

- 夜莺不语的个人/工具网站，可以承载多个分类。夜莺不语是我的名字，网站用于承载我写的工具、博客、我的创作信息。
- 整体网站名字为：夜莺不语 / Nightingale Silence，工具页的后缀也标这个
- 一整套网页项目用同一种前端Vue，使用Vue Router，实现网页构建，丝滑切换网页
- 后续可能会为其他游戏写工具，暂时保留扩展余地

## FFXIV需求

- 当前第一阶段的需求：最终幻想14游戏工具。以还原《最终幻想14》游戏内各类数据为主的工具网页，依赖于对游戏客户端的解构，需要尽可能地还原游戏内体验，数据必须严谨，域名为#/ffxiv
- ffxiv类的工具依靠不同语言的游戏客户端数据实现多语言切换功能，同时进行少量的界面文字本地化
