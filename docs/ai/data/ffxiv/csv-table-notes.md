## 标注规则

- `优先级` 可填：高 / 中 / 低 / 待确认。
- `相关工具` 可填：NSGlamour / NSPlate / NSArmoire / 未来工具名。
- `用途备注` 先写你确定知道的；不确定的保持空白。
- 不要在这里写临时猜测。如果只是猜测，写到 `待确认`。
- 如果实现代码依赖某个列索引，必须在用途备注或模块文档里写明列名/列号和验证来源。
